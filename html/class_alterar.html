Detailed Description

Classe Alterar, é a Interface Gráfica responsável por alterar informações dos funcionários já existentes no Banco de Dados employees.

A Classe Alterar consiste em uma QMainWindow e recebe o nome de um funcionário para que suas outras informações possam ser alteradas.

Bug:
Ao alterar um funcionário já existente no Banco de Dados, é necessário reiniciar o projeto para que a alteração apareça na employeesView (tabela de Funcionários).
