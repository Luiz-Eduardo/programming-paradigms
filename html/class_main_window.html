Detailed Description

Classe MainWindow, é a Interface Gráfica responsável por logar no Software da empresa.

A Classe MainWindow consiste em uma QMainWindow e possui uma página de Login onde as informações serão tratadas e o usuário será enviado para o local apropriado de acordo com seu cargo na empresa.
