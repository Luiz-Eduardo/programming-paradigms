Detailed Description

Classe Caixa é a Interface Gráfica responsável pelas vendas no caixa da empresa.

A Classe Caixa consiste em uma QMainWindow que receberá os dados de compra de todos os clientes da empresa, além disso, a classe é capaz de realizar o update - com o auxílio da classe Conexao - para alterar os valores do Banco de Dados Estoque da tabela products
