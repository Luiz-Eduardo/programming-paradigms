Detailed Description

Classe Financas é a Interface Gráfica responsável pelas Finanças da empresa.

A Classe Financas consiste em uma QMainWindow que mostrará os dados de pagamento de funcionários, as despesas com produtos e receberá o valor das despesas externas, tais como:

  1. Impostos
  2. Manutenção na empresa
  3. Reformas
  4. Contas de energia, luz, aluguel...
