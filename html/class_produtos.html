Detailed Description

Classe Produtos, é a Interface Gráfica responsável por adicionar novos Produtos no estoque da empresa.

A Classe Produtos consiste em uma QMainWindow e possui todos os dados importantes de um produto, que serão colocados no Banco de Dados Estoque na tabela products.

Todo:
Ao inserir novo produto testar se já não existe produto com o mesmo nome e após isso, testar se o preço de compra, venda e validade são o mesmo, se sim, apenas atualizar a quantidade daquele produto de acordo com o valor inserido, se não inserir como produto novo com novo código.
